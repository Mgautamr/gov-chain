# GovChain

GovChain is a full-stack blockchain + AI document governance platform built with Hardhat, Solidity, Express, and React.

## Workspaces

- `contracts/`: Solidity smart contracts
- `backend/`: Express API and blockchain integration
- `frontend/`: React client

## Scripts

- `npm install`
- `npm run contracts:compile`
- `npm run contracts:test`
- `npm run backend:dev`
- `npm run frontend:dev`
- `npm run dev`

## Environment

Copy [backend/.env.example](backend/.env.example) to `backend/.env` and fill in the required values before running the backend against a live or local chain.
